Аналіз Задачі



Задача полягає в пошуку та запису інформації стосовно існуючих вакансій. Для прикладу візьмемо сайт Work.ua https://www.work.ua/ru/jobs/3724496/.

У даних колонках ми можемо побачити опис вакансії та необхідні навички для початку роботи.

Це є данні на веб сайті, які звичайним чином опрацювати не вийде. Тому для роботи на знадобится деякий функціонал для аналізу html документа.

Для мови C існує бібліотека Gumbo, яка призначена для парсингу html документів на C. Наша задача стоїть в проекції аналізу документу, який знаходиться на віддаленому сервері. Для того щоб його отримати для аналізу ми можемо використати бібліотеку cpr, яка призначена для задання http запитів.

Отримавши документ у нас постає задача з записом цих даних у файл тому ми будемо використовувати бібліотеку fstream.

У результаті ми отримуємо алгоритм :

http запит для отримання файлу -> аналіз його вмісту за допомогою gumbo->запис результату у файл
